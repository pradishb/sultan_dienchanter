# Sultan Disenchanter
League of Legends tool to open loots, disenchant, redeem loots, buy champions in batch for multiple accounts.

## Dependencies
1. Python 3.6.8
2. pip
3. pipenv

### Install pipenv
    pip install pipenv

### Create a virtual environment
    pipenv --python 3.6

### Install dependencies
    pipenv install

### Run
    pipenv run gui.py
